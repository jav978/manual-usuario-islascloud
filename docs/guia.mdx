## Manual de usuario Islas Cloud
### Introducción
> Bienvenido al manual de usuario de **Islas Cloud**, la plataforma de reportes en tiempo real que captura información de sensores flexy. Este manual le guiará en el uso y configuración del sistema para sacar el máximo provecho.

### Requisitos del sistema
1. Computadora con Windows 7 o superior, o MacOS 10.12 o superior o linux cualquier distribución
2. Conexión a internet de alta velocidad
Navegador web (recomendado Chrome o Firefox)
3. Cuenta de usuario creada en la plataforma

### Funcionalidades principales
Captura de datos en tiempo real de sensores flexy
Tableros personalizables con gráficos e indicadores
Alertas y notificaciones configurables
Generación de reportes automáticos y a demanda
API de integración con otros sistemas
Rol de administrador para gestionar usuarios y configuraciones
