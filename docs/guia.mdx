## Manual de usuario Islas**Cloud**
### Introducción
> Bienvenido al manual de usuario de **IslasCloud**, la plataforma de reportes en tiempo real que captura información de sensores flexy. Este manual le guiará en el uso y configuración del sistema para sacar el máximo provecho.

> Este manual de usuario por objeto, proporcionar al usuario de la plataforma **IslasCloud Mobile**, la guía necesaria para utilizar el sistema, de una manera amigable y sencilla con los procedimientos adecuados, bajo un esquema de imágenes con las capturas de pantalla, donde se mostrará visualmente el entorno principal, los diferentes módulos y las configuraciones de la aplicación.

### Requisitos del sistema
1. Computadora con Windows 7 o superior, o MacOS 10.12 o superior o linux cualquier distribución
2. Conexión a internet de alta velocidad
Navegador web (recomendado Chrome o Firefox)
3. Cuenta de usuario creada en la plataforma

### Funcionalidades principales
> Captura de datos en tiempo real de sensores flexy
Tableros personalizables con gráficos e indicadores
Alertas y notificaciones configurables
Generación de reportes automáticos y a demanda
API de integración con otros sistemas
Rol de administrador para gestionar usuarios y configuraciones
